Good Quotes- Android app


I developed this simple android application (under the alias BlackFruitApps) using the API provided by Forismatic.com. The application provides a simple user interface to read a random quote. If a user likes it, they can save it on their phone by clicking on the save quote button. I used Android's database(SQLite) to save all the quotes. The user will have the option to delete the saved quote as well. If a user likes a quote they can share the quote either as a text or as a screenshot using a built-in quote capture button.

FindOut-The Sentiment Analysis App


I along with four other teammates developed a Sentiment Analysis Android application. The application would read people's tweets regarding each presidential candidates. We would run our analysis on those tweets and would try to find out whether the tweet is positive, negative or a neutral(using NLTK library on back end). We would then render the result from the back end and show the data in terms of graphs (pie chart for positive, negative and neutral tweets and line chart to show his positive trend over a period). Also we would generate a word cloud(Used Amazon S3 to store a wordcloud for each day).
